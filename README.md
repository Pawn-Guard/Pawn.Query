# Pawn.Query

Pawn.Query - server side plugin that allows you to intercept and modify the samp query mechanism

---

### **What exactly does the plugin intercept**

##### The plugin can intercept queries such as

- Players List
- Server Information
- Detailed Player List
- Server Rules
- Ping
